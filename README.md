# My Finance MVP

Aplicativo web inicial para controle de contas mensais, despesas, receitas, empresas, categorias, metas, relatórios, backup e exportação CSV.

## Como rodar

1. Instale o Node.js
2. Abra a pasta no VS Code
3. Rode:

```bash
npm install
npm run dev
```

Depois abra o link que aparecer no terminal.

## Como publicar

Você pode publicar este projeto na Vercel ou Netlify.
